# 个人助手项目



## 项目背景

因公司项目进度缓慢，空余时间太多，决定敲个项目练练手。又考虑简单的学习了react框架之后没有一个react项目，所以就有了该项目。



## 项目说明

一个基于reacr-create-app搭建的移动端项目和基于node.js搭建的后台



### 适合人群

适合本人这样刚入门react的新手



### 技术栈

> react
>
> react-create-app
>
> axios
>
> moment.js
>
> Ant Design Mobile
>
> echarts
>
> echarts-for-react 
>
> mysql
>
> express
>
> express-jwt
>
> jsonwebtoken



### 运行地址

```js
// 前端
http://127.0.0.1:3000

// 后台
http://127.0.0.1:3001
```



### 主要模块

主要模块包括

1. 登录模块
2. 今日事项模块
3. 今天吃什么九宫格抽奖模块
4. 备忘录模块
5. 记账本模块
6. 更换皮肤模块
7. 设置模块（暂时还没有完成...）
